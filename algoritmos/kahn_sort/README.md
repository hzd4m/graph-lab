# Kahn Sort

O **Kahn Sort** é um algoritmo clássico de ordenação topológica para grafos direcionados acíclicos (DAGs).

## Status atual

A implementação ainda **não foi criada**. Este módulo está reservado para construção posterior durante as etapas didáticas do projeto.

## Arquivos da pasta

- `kahn_sort.py`: placeholder para o código principal;
- `exemplos.py`: placeholder para exemplos executáveis;
- `entrada_customizada.py`: placeholder para entradas definidas pelo usuário;
- `casos/dag_simples.json`: modelo inicial de DAG simples;
- `casos/ciclo_simples.json`: modelo inicial com ciclo;
- `testes/test_kahn_sort.py`: placeholder para testes automatizados.

## Casos a serem estudados futuramente

- DAG simples para validar fluxo básico;
- grafo com ciclo para observar comportamento em cenário inválido para ordenação topológica.
