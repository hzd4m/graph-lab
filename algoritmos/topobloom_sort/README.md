# TopoBloom Sort

O **TopoBloom Sort** é uma extensão autoral do Kahn Sort, proposta para fins didáticos no **graph-lab**.

## Ideia pedagógica

A proposta é combinar ordenação topológica com uma preferência pedagógica inspirada na Taxonomia de Bloom, para apoiar sequenciamento de estudo.
